С помощью данного скрипта можно совершать исходящие звонки на номера в РБ. Работает с помощью API мобильного
оператора A1. Чтобы совершить звонок нужно скопировать номер телефона в буфер обмена и нажать сочетание
клавиш CTRL + ALT. Звонки совершаются с того номера, который сохранен в файле config.pkl, его можно поменять,
для этого нужно ввести номер телефона в поле и нажать "Применить", после этого появляется сообщение о том,
что номер телефона изменен. Номер телефона с которого происходят звонки  должен принадлежать компании, которая
получила доступ к API.
